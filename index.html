O Amazon S3 Explorer é um aplicativo que usa o AWS JavaScript SDK no navegador para facilitar a visualização do conteúdo de um bucket do S3 em um navegador da web.

Para fazer upload de arquivos, primeiro navegue até o bucket & pasta de sua escolha e arraste & solte arquivos na tabela principal do S3 Explorer.

Para excluir arquivos, primeiro escolha os arquivos que deseja excluir clicando nas caixas de seleção na coluna Selecionar e, em seguida, clique no ícone Lixeira. Você será solicitado a confirmar esta ação.

Para mais detalhes e para baixar o código-fonte, consulte nosso GitHub página.

O JavaScript no navegador não pode exibir o conteúdo de um bucket do Amazon S3, a menos que o bucket tenha a configuração apropriada de compartilhamento de recursos entre origens (CORS).

Para configurar o CORS, você cria uma configuração CORS que identifica as origens permitidas para acessar seu bucket e as operações (métodos HTTP) suportadas.

Para fazer isso, selecione seu bucket no painel de buckets do Amazon S3 Console e clique para revelar Permissions no painel Properties. Clique em Editar configuração do CORS. O painel CORS Configuration Editor será aberto com um campo onde você pode inserir uma configuração CORS. Insira uma configuração semelhante à seguinte: 

<?xml version="1.0" encoding="UTF-8"?>
<CORSConfiguration xmlns="http://s3.amazonaws.com/doc/2006-03-01/">
    <CORSRule>
        <AllowedOrigin>*</AllowedOrigin>
        <AllowedMethod>GET</AllowedMethod>
        <AllowedMethod>POST</AllowedMethod>
        <AllowedMethod>PUT</AllowedMethod>
        <AllowedMethod>HEAD</AllowedMethod>
        <MaxAgeSeconds>3000</MaxAgeSeconds>
        <AllowedHeader>*</AllowedHeader>
        <ExposeHeader>ETag</ExposeHeader>
        <ExposeHeader>x-amz-meta-myheader</ExposeHeader>
    </CORSRule>
</CORSConfiguration>

Por favor, veja o projeto README para obter detalhes mais específicos sobre como criar uma configuração CORS apropriada.


Aqui está um exemplo de política de bucket do S3 que você pode usar. Mudar 999999999999 para o número da sua conta da AWS e altere meubucket para o nome do seu balde.

{
    "Version": "2012-10-17",
    "Statement": [{
        "Sid": "AllPermissionsForOwner",
        "Effect": "Allow",
        "Principal": {
            "AWS": "arn:aws:iam::999999999999:root"
        },
        "Action": "s3:*",
        "Resource": "arn:aws:s3:::mybucket/*"
    }]
}

Por favor, veja o projeto README para obter detalhes mais específicos sobre como criar uma política de bucket S3 apropriada.
